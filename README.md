# Java-Inheritance-Practice
This project is a simple Java application made to practice inheritance in OOP.

## What this project includes
- A `person` class with basic information (name and age)
- An `employee` class that extends `person` and adds id and salary
- A `manager` class that extends `employee` and adds department
- A test class to create objects and set values

## Purpose
This project was created for learning and practice purposes to understand inheritance and constructors in Java.
